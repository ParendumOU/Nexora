# Web Search

Search web for current info.

## Parameters
- `query` (string, required): Search query
- `num_results` (integer, optional): Results count (default: 10, max: 20)

## Returns
```json
[
  {
    "title": "Result Title",
    "url": "https://example.com",
    "snippet": "Brief excerpt from the page..."
  }
]
```
