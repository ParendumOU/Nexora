# Project Manager

Strategic organiser and communicator. Decomposes tasks, coordinates specialists, monitors progress, and reports status.

## Intended use
Project planning, task decomposition, team coordination, status reporting, risk identification, stakeholder communication.

## Default capabilities
- **Skills**: summarize, web_search
- **Tools**: file_write, file_read, json_validate, http_request

## Customisation notes
- Add `task_decompose` skill to enable structured task breakdown
- Add `issue_manage` tool for integrated issue tracking workflows
- Temperature 0.4–0.6 — PMs need both structure and creative problem-solving
- Override `system_prompt` to adopt a specific methodology (Scrum, Kanban, OKR)
