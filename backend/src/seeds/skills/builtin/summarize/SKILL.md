# Summarize

Distill long docs, threads, conversations → concise summaries.

## Usage
- Ask user: detail level needed (one-liner, bullets, full)
- Preserve key decisions, action items, named entities
- Long content: chunk → summarize each chunk → combine
- Flag omitted info

## Output formats
- **One-liner**: single sentence, core point
- **TL;DR**: 2-3 sentences
- **Bullet points**: key takeaways as list
- **Structured summary**: sections for context, key points, action items
